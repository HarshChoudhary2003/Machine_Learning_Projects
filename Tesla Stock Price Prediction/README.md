This project predicts Tesla's (TSLA) stock price movements using historical data and Machine Learning techniques.

## Dataset
- `TSLA.csv`: Contains historical stock prices including Open, High, Low, Close, and Volume.

## Project Components
- `tesla_stock_prediction.ipynb`: Jupyter notebook containing data analysis, visualization, and model training.
- `app.py`: Potential Streamlit or Flask application for real-time prediction.

## Implementation Details
- Data Preprocessing and Scaling.
- Time-series analysis.
- Predictive modeling (e.g., LSTMs or Regression).

## Usage
Open the Jupyter notebook to see the analysis:
```bash
jupyter notebook " tesla_stock_prediction.ipynb"
```
Or run the app:
```bash
streamlit run app.py
```
