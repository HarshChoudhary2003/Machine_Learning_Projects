## ğŸš€ Getting Started

### Prerequisites
- Python 3.8 or higher
- pip package manager

### Installation

```bash
# Clone the repository
git clone https://github.com/HarshChoudhary2003/Machine_Learning_Projects.git
cd Machine_Learning_Projects

# Create virtual environment
python -m venv venv
source venv/bin/activate
```

